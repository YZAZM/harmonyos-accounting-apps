## 使用建议

### 项目经理
1. 先阅读 `TESTING_AND_RELEASE_PLAN.md` 了解整体规划
2. 使用 `GO_LIVE_CHECKLIST.md` 跟踪项目进度
3. 参考 `QUICK_REFERENCE.md` 掌握关键节点

### 测试工程师
1. 使用 `TEST_CASES.md` 执行测试
2. 记录测试结果在测试用例文档中
3. 及时反馈Bug给开发团队

### 开发人员
1. 参考 `TEST_CASES.md` 进行自测
2. 配合测试团队修复Bug
3. 按照 `GO_LIVE_CHECKLIST.md` 准备上线

### 产品经理/运营
1. 使用 `APP_STORE_MATERIALS.md` 准备上架材料
2. 撰写应用描述和宣传文案
3. 协调设计和开发完成素材准备

### UI设计师
1. 参考 `APP_STORE_MATERIALS.md` 中的图标规格
2. 设计应用图标和宣传图
3. 协助准备应用截图
